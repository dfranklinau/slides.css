slides.css

A CSS framework for creating PDF presentations.

What does slides.css do?

slides.css is a HTML and CSS alternative to traditional slideshow software.

It utilises a browser's "Save to PDF" capabilities and CSS positioning to create PDF presentations.

Features

slides.css offers somewhat opinionated typographic styles for headings, paragraphs, lists and code blocks.

It also includes styling for visual figures, text and slide alignment, and headers and footers.

Requirements

All slides should match the following HTML structure:

<section class="slide">
  <!-- slide HTML -->
</section>
