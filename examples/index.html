slides.css

A CSS framework for creating PDF presentations.

slides.css is a HTML and CSS alternative to traditional slideshow software.

It utilises a browser's "Save to PDF" capabilities and CSS positioning to create static PDF presentations.

By using HTML, traditional slideshows can be created in a highly accessible format, structured with headings and other semantic elements.

Use PDF for visual presentations, then distribute the HTML for assistive technologies.

slides.css offers the following features:

  • fixed and non-fixed header and footer;
  • opt-in typography styling;
  • basic figure and figure caption styles;
  • page numbering (with JavaScript); and
  • configuration through CSS custom properties.
